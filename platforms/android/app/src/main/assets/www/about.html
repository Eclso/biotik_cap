BIOTIK - Version : 1.1.0

Western Ghats Trees ID is a graphical identification system for the trees species of the Western Ghats of India one of the biodiversity hotspots of the world. The IDentification Assistée par Ordinateur (IDAO)was implemented to generate identification key for tree species under the Project "Biodiversity Informatics and co-Operation in Taxonomy for Interactive shared Knowledge base (BIOTIK) ".

This system, entirely graphic, enables specialists and non-specialists to make determinations with the help of just a few number of characters, chosen according to their availability for the particular tree to be identified. The use of vegetative characters, as well as an extensive illustration of species, is the mainstay of the application. Most of the characters of the used in the tool can be observed with the naked eye. The user friendly identikit was developed with the two interface viz., graphical interface and result interface. The present version acquire the result pages with images from the http://www.ifpindia.org/digitaldb/online/biotik/ . This key allows the user to identify 528 tree species.
